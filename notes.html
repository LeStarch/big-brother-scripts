To create a new monitor station
  1. Install Fedora

    Other distros may work but Fedora has support for the newest hardware drivers and has been the easiest solution. Keep the stations consistent.

    • Set the root password to the standard
    • Create an admin user 'monitor' as fullname 'Scale AV team'
  2. Set up the host
    1. Turn on the SSHD server 
      sudo systemctl enable sshd.service
sudo systemctl start sshd.service
    2. Set up the desktop
      1. Arrange the monitors:
      2. Right click on the screen background and choose 'settings'
      3. Select 'Displays'
      4. Click on 'Arrange Combined Displays'
      5. Click on 'Apply'
      6. Click on 'Accept changes'
      7. Click the back arrow in the upper left
      8. Select 'Power'
      9. Change the 'blank screen' time to 15 minutes
      10. Set the hostname 
        sudo vi /etc/hostname
        monitor<number>.scaleav.us
      11. Reboot to pick up the new hostname
    3. Set up access
      1. Create a ssh key 
        ssh-keygen
      2. Copy the .ssh/id_rsa.pub onto the end of all other station .ssh/authorized_keys files.
      3. Copy those workstation's id_rsa.pubs onto the end of ~monitor/.ssh/authorized_keys.
    4. Set up name resolution
      1. Edit the hosts file 
        sudo vi /etc/hosts
      2. Get our ip address 
        ip addr
      3. Add our IP address to all the other stations' hosts files.
    5. Copy in the software 
      cd
git clone https://github.com/socallinuxexpo/av-video_monitor.git scaleav
git clone https://github.com/socallinuxexpo/follow-sound.git scaleav/src/follow_sound
    6. Install pre-requisite software and set up configuration files 
      cd ~/scaleav
./setup_monitor
    7. Launch the video monitoring suite 
      cd ~/scaleav
./start_monitor
